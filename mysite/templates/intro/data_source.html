활용한 빅데이터 분석기법

1. 데이터 전처리 (Data Preprocessing)

엑셀을 이용한 데이터 가공:

  • 데이터 정리 및 정제: 엑셀에서 데이터를 정리하고, 결측값 처리, 중복 데이터 제거, 잘못된 형식 수정 등의 작업을 진행했습니다.
  • 데이터 집계 및 변환: 엑셀에서 다양한 연산(예: 평균, 합계, 그룹화)을 통해 데이터를 집계하거나 변환하는 기법을 사용했습니다. 예를 들어, 시간대별, 지하철역별 혼잡도를 계산하고, 추가적인 지표를 생성했습니다.

2. 탐색적 데이터 분석 (Exploratory Data Analysis, EDA)

엑셀을 이용한 데이터 탐색:

  • 기초 통계 분석: 엑셀에서 데이터의 분포를 파악하고, 평균, 표준편차, 최대값, 최소값 등을 계산하여 데이터를 분석했습니다.
  • 데이터 시각화 (기본 차트 사용): 엑셀에서 기본적인 시각화(예: 막대 그래프, 선 그래프, 히스토그램 등)를 이용해 데이터의 패턴을 분석하고 시각적으로 표현했습니다.

3. 데이터 시각화 (Data Visualization)

태블로에서 시각화:

  • 대시보드 및 인터랙티브 시각화: 태블로에서 대시보드를 구성하고, 다양한 필터와 인터랙티브 요소를 추가하여 사용자가 쉽게 데이터를 분석할 수 있도록 만들었습니다.
  • 지리적 시각화 (Geospatial Visualization): 지하철역 및 지역별 데이터를 시각화할 때, 태블로에서 지리적 시각화 기법을 활용하여 지역별 혼잡도나 재개발 지역을 지도에 표시했습니다.
  • 패턴 인식 및 트렌드 시각화: 혼잡도, 시간대, 지역 등을 비교하면서 패턴 인식 및 트렌드 시각화 기법을 사용하여 분석했습니다.

4. 집계 및 그룹화 (Aggregation and Grouping)

엑셀에서의 데이터 집계:

  • 피벗 테이블 (Pivot Table): 엑셀에서 피벗 테이블을 사용하여 데이터를 그룹화하고, 이를 기반으로 교차 분석을 진행했습니다. 예를 들어, 지하철역별, 시간대별, 지역별로 혼잡도를 계산했습니다.

5. 가설 설정 및 인사이트 도출

패턴 발견:

  • 가설 설정 및 분석: 엑셀과 태블로에서의 분석을 통해 혼잡도 패턴이나 출퇴근 시간대, 재개발 지역과의 관계를 도출하고 이를 분석하여 가설을 설정하고 인사이트를 도출했습니다. 예를 들어, "특정 지역의 아파트 재개발이 완료되면 혼잡도가 증가한다"는 가설을 설정하고 이를 분석하여 확인하는 과정입니다.

6. 데이터 모델링 기법 (제한적)

단순 분석 기법 사용:

  • 통계적 분석: 고급 모델링 기법을 사용하지 않았지만, 데이터를 단순화하여 분석하거나 통계적 분석을 통해 혼잡도를 예측하는 방식이 간접적으로 활용되었습니다. 예를 들어, 평균값, 표준편차 등을 사용하여 혼잡도를 비교하는 작업이 포함될 수 있습니다.
